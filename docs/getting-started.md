# 书舟 ReadFlow v0.1 — 用户指南

> 部署在你自己硬盘里的私人阅读服务。书放在本地，浏览器里看。

---

## 快速开始

### 1. 安装依赖

需要一个 Python 环境（推荐 3.11+）。在项目目录里执行：

```bash
# 安装 uv（如果还没有）: curl -LsSf https://astral.sh/uv/install.sh | sh
uv sync
```

这会在项目内创建 `.venv/` 虚拟环境，**不会污染系统 Python**。

### 2. 启动服务

```bash
uv run uvicorn app.main:app --port 8765
```

服务启动后，打开浏览器访问：

```
http://localhost:8765
```

### 3. 放入你的第一本 epub

把 `.epub` 格式的电子书复制到项目根目录的 `books-library/` 文件夹下：

```bash
# 示例
cp ~/你的书/北京法源寺.epub books-library/
```

书舟会自动检测新书，几秒钟后刷新首页即可看到。

## 日常使用

### 书库首页

打开 `http://localhost:8765` 会看到所有已入库的书籍，以封面卡片形式展示：

- **封面** — 从 epub 自动提取，如无封面则显示占位
- **书名 / 作者** — 从 epub 元数据自动解析
- **阅读进度条** — 显示上次读到百分之几

支持按照 **最近添加** 的顺序排列（v0.2 将增加更多排序方式）。

### 打开一本书

点击任意书籍封面，进入阅读器页面。

### 翻页方式

| 操作 | 效果 |
|---|---|
| **← 左箭头键** | 上一页 |
| **→ 右箭头键 / 空格键** | 下一页 |
| **点击屏幕左侧 40%** | 上一页 |
| **点击屏幕右侧 40%** | 下一页 |
| **点击屏幕中间 20%** | 不翻页（方便用鼠标选文字） |

### 选中文本 — 划线 / 复制

**用鼠标选中一段文字**，底部会自动弹出工具栏，两个按钮：

- **划线** — 高亮这段文字，自动保存。刷新页面后仍可见
- **复制** — 将选中文本复制到系统剪贴板

划线数据会自动同步到数据库，下次打开同一本书时，你的历史划线会重新显示。

### 目录导航

点击顶部工具栏的 **☰ 目录按钮**，输入章节序号可快速跳转。

### 排版设置

点击顶部工具栏的 **「文」按钮**，右侧滑出排版面板，可实时调整：

| 设置 | 范围 | 说明 |
|---|---|---|
| **字号** | 12–28 px | 拖动滑块，下方预览行同步变化 |
| **行距** | 1.2–2.8 | 段落行间距 |
| **页边距** | 窄 / 中 / 宽 | 段落左右留白 |
| **字体** | 霞鹜文楷 | 内置中文字体（OFL 协议） |

设置即时生效，自动保存在浏览器本地（localStorage），刷新或重开同一本书时自动恢复。每项设置都会覆盖电子书自带的排版样式。

## 阅读进度

书舟会自动保存你的阅读位置：

- **当前章节位置** — 精确到段落
- **全书阅读百分比** — 显示在顶部进度文本

关闭浏览器、换设备（同一局域网内访问同一 URL）都能从上次读到的地方继续。

> 注意：进度使用 CFI（基于内容流的绝对定位），不受字号/行距/屏幕宽度影响，跨设备/跨排版精准恢复。

## 文件监听说明

书舟使用 watchdog 实时监听 `books-library/` 目录：

- **新放入**的 epub 自动入库
- **重命名/移动**的同名文件会被识别为同一本书，不重复入库
- 放入非 epub 格式的文件（如 .pdf）会标记为"不支持"，不会破坏书库

> 如果书放进去了一段时间还没显示，可能是文件较大、解析需要时间，请稍等后刷新。

## 常见问题

**Q: 为什么有些书没有封面？**
> epub 文件本身未内嵌封面图，或封面命名方式不规范。书舟会尽可能从 epub 中提取，失败时显示占位。

**Q: 为什么页面是白色/深色模式不生效？**
> 目前 v0.1 的阅读器支持系统的深色模式偏好（`prefers-color-scheme: dark`）。确保你的系统/浏览器已开启深色模式。

**Q: 同一本书出现两次？**
> 正常不会。书舟使用文件内容 hash 去重，即使改名或移动位置也不会重复。如遇重复，请检查是否为两个不同文件。

**Q: 如何备份阅读数据？**
> 所有书籍元数据、阅读进度和划线都保存在 `data/readflow.db` 这一个 SQLite 文件中。直接复制这个文件即可备份。书籍原文件（epub）保存在 `books-library/` 目录下，不随数据库移动。

## 已知限制（v0.1）

- 仅支持 epub 格式
- 仅支持单用户，无多账号
- 无全文搜索（v0.2 引入）
- 无外部元数据（豆瓣评分/简介），v0.2 计划接入
- 无 AI 功能（每日总结/推荐/翻译），后续版本引入

## 停机与重启

```bash
# 停止：按 Ctrl+C
# 重新启动
uv run uvicorn app.main:app --port 8765
```

重启后所有阅读进度和划线数据都在，不会丢失。
